# AI Prompt Template (Codex)

Use this template for every task.

## 1) Task

What needs to be implemented or fixed.

## 2) Scope

What is in scope and out of scope.

## 3) Constraints

1. Do not change unrelated files.
2. Do not change behavior outside specified module/page.
3. Keep backward compatibility where required.

## 4) Allowed Files

List exact files/directories that can be edited.

## 5) Forbidden Files

List files/directories that must not be touched.

## 6) Required Validation

List required checks/tests and expected outputs.

## 7) Output Format

Require:

1. Summary of changes
2. File list
3. Risk notes
4. Validation evidence

## AI Safety Rules

1. Never include credentials/secrets.
2. Never bypass review gates.
3. Never assume merge approval.
4. Always ask for clarification if scope is ambiguous.
